# Project Ganache
Design your own wedding cake.

# Author

## Run
* open "Terminal"
* cd into p4/
* enter "python -m SimpleHTTPServer"
* open "http://localhost:8000/"

## Shortcuts
* d - Delete
* z - Undo delete

## Libraries
* LESS
* jQuery
* jQuery UI

## Features
* Choose size
* Choose flavor
* Add decorations
* Delete decorations
* Undo delete operations
